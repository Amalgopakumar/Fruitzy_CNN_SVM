# 🍎 Real-Time Fruit Grading System using CNN-SVM

## 📌 Overview

Post-harvest fruit quality inspection is a major challenge in agricultural supply chains due to inconsistent manual grading, labour dependency, and slow processing speed.

This project presents a **real-time non-destructive fruit grading system** that combines **Machine Vision** and **Deep Learning** to automatically classify fruits into quality grades.

The system supports:

* Mango 🥭
* Apple 🍎
* Banana 🍌

Each fruit is classified into:

* Premium
* Good
* Rotten

---

## 🚀 Key Features

✅ Real-time fruit grading on conveyor belts
✅ Non-destructive quality inspection
✅ Hybrid CNN + SVM model
✅ Supports multiple fruit types
✅ Flask monitoring dashboard
✅ High-speed inference (<100 ms per fruit)
✅ 96.8% classification accuracy

---

## 🧠 Model Architecture

### CNN as Feature Extractor

* Convolution Layers
* Max Pooling Layers
* Batch Normalization
* Dense Feature Representation

### SVM Classifier

* RBF Kernel
* Optimized decision boundaries
* Final quality prediction

---

## 📂 Dataset

* 12,000+ Fruit Images
* RGB High-resolution images
* Data Augmentation:

  * Rotation
  * Flip
  * Zoom
  * Brightness Variation
